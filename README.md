# 💳 Alke Wallet Database

Proyecto desarrollado para el módulo:
Fundamentos de Bases de Datos Relacionales.

---

## 📌 Descripción

Diseño e implementación de una base de datos relacional para una Wallet digital.

La base de datos permite:

- Gestión de usuarios
- Gestión de monedas
- Registro de transacciones
- Consultas SQL
- Integridad referencial
- Uso de DDL y DML
- Transacciones ACID

---

## 🛠️ Tecnologías utilizadas

- SQL
- MySQL 8
- SQLOnline
- Visual Studio Code
- GitHub
